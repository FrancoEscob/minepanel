# Roadmap por fases

## Fase 0 — Diseño y preparación (1-2 semanas)
- Bootstrap de stack aprobado (Next.js + backend TypeScript + Prisma).
- Definir contratos de dominio (`NodeProvider`, `Telemetry`).
- Montar estructura base del repo.

## MVP — Operación local completa (6-8 semanas)
- Auth local + roles básicos.
- Wizard de creación Vanilla/Paper.
- Ciclo de vida servidor + consola real-time.
- Configuración base y listas de jugadores.
- Backups locales + restore.
- Monitor básico.

## V1 — Productividad y seguridad (6-10 semanas)
- Importar servidor existente.
- Jobs avanzados y notificaciones.
- Gestión de mundos mejorada.
- Integración Modrinth.
- Auditoría más completa y RBAC extendido.

## V2 — Escala y ecosistema
- Arquitectura multi-nodo con agents.
- Integración CurseForge (cumplimiento legal/API).
- Safe update mode con rollback automatizado.
- Métricas avanzadas y alertas inteligentes.
